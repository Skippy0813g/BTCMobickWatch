# BTCMobickWatch

**개인키 입력 없이** BTC 계열(Bitcoin·BTCMobick·LaptopMining)과 EVM 자산을 조회할 수 있는 watch-only 안드로이드 지갑앱입니다.<br>
고급 사용자들이 콜드월렛과 QR로 연동해 BTC 계열 자산을 에어갭 방식으로 관리할 수 있습니다.

## 보안 원칙

- **Watch-only** — 이 앱은 개인키·니모닉을 입력받지도 보관하지도 않습니다.
- **기기 보안 강제** — 지문/PIN 등 기기 잠금이 있어야 실행되며, 진입 시마다 기기 소유주 인증을 요구합니다.
- **하드웨어 기반 저장** — 지갑 정보는 기기 보안 영역(Keychain)에 암호화 저장되고, 기기 밖으로 나가지 않습니다.

## 지원 자산

- **BTC 계열**: Bitcoin(BTC), BTCMobick(BMB), LaptopMining(LTM) — 조회 및 에어갭 송금 지원
- **EVM**: BNB·Ethereum·Base 및 관련 토큰(WBMB·MOVN·USDT 등) — 현재 조회 전용
